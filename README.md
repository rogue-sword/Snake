# Snake
A Snake game made using C++ &amp; SFML.

### Information
- The actual code is inside the project folder.
- You will need: git cmake ninja.
- You will also need clang-format if you wish to use the format option provided in the makefile.
- The images are embeded in the code for creating standalone binary.
- The image arena was created by me &amp; logo was taken from svg-repo and is present under the CC0 License.
